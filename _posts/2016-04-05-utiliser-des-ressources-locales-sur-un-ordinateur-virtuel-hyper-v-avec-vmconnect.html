Vous pouvez permettre à un ordinateur virtuel d'accéder aux ressources locales d'un ordinateur, comme un lecteur flash USB amovible, quand vous utilisez le gestionnaire de connexion à un ordinateur virtuel (VMConnect).Pour ce faire, activez le mode de session étendu sur l'hôte Hyper-V, utilisez VMConnect pour vous connecter à l'ordinateur virtuel et, avant de vous connecter, choisissez la ressource locale à utiliser.

Activer le mode de session étendu sur l'hôte Hyper-V

Si votre hôte Hyper-V exécute Windows 8 ou Windows 8.1, il n'est peut-être pas nécessaire d'exécuter la procédure suivante pour activer le mode de session étendu.Il est activé par défaut.En revanche, si votre hôte exécute Windows Server 2012 ou Windows Server 2012 R2, vous devez activer le mode de session étendu pour l'utiliser.Il est désactivé par défaut pour ces systèmes d'exploitation.

Pour activer le mode de session étendu

  1. Connectez-vous à l'ordinateur qui héberge l'ordinateur virtuel.
  2. Dans le Gestionnaire Hyper-V, sélectionnez le nom d'ordinateur de l'hôte.

  3. Sélectionnez Paramètres Hyper-V.

  4. Sous Serveur, sélectionnez Stratégie de mode de session étendu.

  5. Cochez la case Autoriser le mode de session étendu.

  6. Sous Utilisateur, sélectionnez Mode de session étendu.

  7. Cochez la case Autoriser le mode de session étendu.
  8. Click
Choisir la ressource locale à utiliser

Vous pouvez choisir une ressource locale, comme une imprimante, le Presse-papiers ou un lecteur local sur l'ordinateur que vous utilisez pour vous connecter à l'ordinateur virtuel.

Pour sélectionner une ressource locale comme un lecteur

  1. Ouvrez VMConnect.
  2. Sélectionnez l'ordinateur virtuel auquel vous souhaitez vous connecter.
  3. Click

  4. Sélectionnez Ressources locales.

  5. Click

  6. Sélectionnez le lecteur que vous souhaitez utiliser sur l'ordinateur virtuel, puis cliquez sur Ok.

  7. Sélectionnez Enregistrer mes paramètres pour les futures connexions à cet ordinateur virtuel.

  8. Click
Modifier les paramètres VMConnect

Vous pouvez facilement modifier vos paramètres de connexion pour VMConnect en exécutant la commande suivante dans Windows PowerShell ou l'invite de commandes :

VMConnect.exe <ServerName> <VMName> /edit

Éléments nécessaires pour utiliser les ressources locales d'un ordinateur

Pour pouvoir utiliser les ressources locales d'un ordinateur sur un ordinateur virtuel :

  • Les paramètres Stratégie de mode de session étendu et Mode de session étendu de l'hôte Hyper-V doivent être activés.
  • L'ordinateur que vous utilisez pour vous connecter à l'ordinateur virtuel avec VMConnect doit exécuter Windows 8, Windows 8.1, Windows Server 2012 ou Windows Server 2012 R2.
  • Les services Bureau à distance doivent être activés sur l'ordinateur virtuel qui doit exécuter Windows Server 2012 R2 ou Windows 8.1 comme système d'exploitation invité.

Si votre ordinateur virtuel répond à ces critères, vous pouvez utiliser toutes les ressources locales suivantes disponibles à partir de l'ordinateur où vous utilisez VMConnect.

  • Configuration de l'affichage
  • Audio
  • Imprimantes
  • Presse-papiers pour copier et coller
  • Cartes à puce
  • Périphériques USB
  • Lecteurs
  • Périphériques Plug-and-Play pris en charge
