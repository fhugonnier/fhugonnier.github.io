Dans cet article, on va découvrir comment contrôler notre box Domoticz par la voix. Pour cela, on va installer Yana-server sur notre pi.

 

Présentation.

Yana est une plateforme domotique pilotable à la voix et on va lui installer un plugin pour l’intégrer à Domoticz.

page du GitHub, le Blog.

 

Installation.

 

On se connecte en SSH à notre pi.

 

Puis on copie la commande suivante pour installer Yana.

sudo apt-get install git-core && sudo apt-get install sqlite3 && sudo apt-get install php5-sqlite && cd /var/www/ && sudo git clone https://github.com/ldleman/yana-server.git /var/www/yana-server && sudo chown -R www-data:www-data yana-server && sudo chown root:www-data /var/www/yana-server/plugins/radioRelay/radioEmission && sudo chmod +s /var/www/yana-server/plugins/radioRelay/radioEmission

Aprés l’installation (5 min),ouvrez votre navigateur et tapez l’adresse de yana:

http://adresse.de.voe.rpi/yana-server

Crée votre compte, connectez vous et rendez-vous dans :

Configuration / Plugins / Market.

 

Recherchez le plugin Domoticz.

Cliquez sur installer, ensuite aller sur l’onglet inactif chercher Domoticz et activé-le.

On va configurer votre plugin, aller sur :

Configuration / Préférence / Serveur Domoticz.

Remplissez le formulaire avec les paramètres pour votre Domoticz et cliquez sur sauvegarder.

Maintenant, il faut configurer vos commandes vocal donc rendez-vous dans:

Configuration / Domoticz.

Toutes vos commande pour actionner vos modules Domoticz sont là.

On clique sur la croix pour pouvoir ajouter une commande.

Toutes les commandes ajoutées vont se retrouver dans l’onglet Commandes vocales.

A partir de là, on peut activer/désactiver une commande, la modifier ou la supprimer.
