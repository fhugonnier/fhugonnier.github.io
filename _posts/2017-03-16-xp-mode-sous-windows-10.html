Il vous faudra télécharger et installer 7Zip: 7-Zip

1) Télécharger le XP Mode: https://www.microsoft.com/fr-fr/down...s.aspx?id=8002

2) Il faut extraire le contenu du pack exécutable du XP Mode
Faites un clic droit sur l'exécutable et cliquer sur Extraire les fichiers

3) Donner le chemin pour l'extraction d'un dossier que vous aurez créé, de préférable sur un deuxième disque dur ou un disque dur USB externe

4) Cliquer sur Fermer une fois l'extraction terminée

5) Créer un nouveau dossier et renommer le XPM

6) Ouvrez le dossier ou a été extrait le XP Mode

7) Ouvrir le dossier sources

8) Extraire le fichier XPM dans le dossier XPM

9) Sélectionner le répertoire XPM, décocher la case ou c'est indiqué et cliquer sur OK

10) Extraction en cours veuillez patienter

11) Ouvrir le répertoire XPM

12) Renommer le fichier VirtualXPVHD en VirtualXPVHD.vhd

13) Ce qui donnera ceci

* À ce stade, vous pouvez supprimer maintenant le premier dossier d'extraction du XP Mode se nommant WindowsXPMode_fr-fr

14) Aller maintenant dans le Panneau de configuration et cliquer sur Programmes

15) Cliquer sur Activer ou désactiver des fonctionnalités Windows

16) Cochez tout ce que vous pouvez qui concerne Hyper-V
Les boîtes à cocher ne doivent pas être grisées
Si une case est grisée il faudra configurer votre bios pour activer l'Hyper-V dans la section du processeur
Une fois les cases cochées cliquer sur OK

17) Application des modifications en cours, veuillez patienter

18) Une fois l'installation terminée vous devez redémarrer pour terminer l'installation

19) Retourner ensuite dans Activer ou désactiver des fonctionnalités Windows
Si vous avez une zone grisée une fois redémarré, vous devrez aller dans le bios activer l'Hyper-V du processeur
Généralement pour accéder au bios au redémarrage il faut appuyer sur la touche F2 ou F8 ou F10 ou Supprimer

20) Vérifier si toutes les fonctionnalités de l'Hyper-V est entièrement activé via l'invite de commande en tant qu'administrateur
Rechercher Invite et exécuter le en tant qu'administrateur

21) Saisissez la commande systeminfo.exe et appuyer sur la touche Retour du clavier

22) Si Non est indiqué dans la configuration d'Hyper-V c'est que la fonctionde cette dernière n'a pas été activée dans la section du processeur

23) Une fois l'Hyer-V activé dans le bios tout sera à Oui

24) Retournez dans le panneau de configuration / Activer ou désactiver des fonctionnalités Windows

25) Cochez la ou les zones qui étaient grisées puis cliquer sur OK

26) Installation des fonctionnalités

27) Redémarrer une fois terminer

28) Rechercher maintenant Hyper et épinglez le au menu Démarrer
Cliquer sur Gestionnaire Hyper-V pour le lancer

29) Sélectionnez le nom de votre PC à gauche

30) Cliquer maintenant sur Nouveau
Puis sur Ordinateur Virtuel

31) Saisissez le nom XP Mode
Puis cliquer sur Suivant

32) Sélectionner Génération 1

33) Indiquer la taille de la mémoire que vous désirez

34) Mise en réseau (pour mo c'est non connecté) et cliquer sur suivant

35) Sélectionner maintenant Utiliser un disque dur virtuel exixtant
Puis cliquer sur Parcourir

36) Sélectionner VirtualXPVHD.vhd
Cliquer sur Ouvrir

37) Cliquer ensuite sur Suivant

38) Cliquer maintenant sur Terminer

39) Par défaut le nouveau XP Mode est désactivé
Il faut l'activer
Faites un clic droit sur ce dernier puis cliquer sur Se Connecter

40) Un tableau noir vous indiquera que le XP Mode est éteint
Cliquer sur le petit bouton Vert pour le lancer

41) Démarrage de l'ordinateur XP Mode en cours

42) Vous devrez suivre les indications pour configurer le système virtuel
Une fois cela fait le XP Mode sera opérationnel
