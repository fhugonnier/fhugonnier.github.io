Voici une petite astuce pour tester facilement et rapidement une connexion vers SQL Server (toutes les versions).

 

  1. Créez un fichier (.txt. par exemple) et renommer le en .udl
  2. Entrez dans les propriétés de ce fichier
  3. Allez directement sur l'onglet Connection
  4. Renseignez les informations de votre instance SQL
  5. Cliquez sur Test Connection
  6. Vous aurez soit un message d'échec, soit un message de succès

Ce test vous permet de valider la connexion à votre instance SQL par rapport au compte définie dans les paramètres. Il faudra ensuite appliquer les bons droits sur le compte pour effectuer les tâches souhaitées.
