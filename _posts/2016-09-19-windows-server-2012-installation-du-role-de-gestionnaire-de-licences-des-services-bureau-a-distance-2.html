Nous allons installer le rôle de gestionnaire de Licences des services bureau à distance (Remote Desktop Services Licenses Manager). Pour ce faire, ouvrir le server manager puis cliquer sur Gérer.

Cliquer sur Ajouter des rôles et fonctionnalités.

Cliquer sur Suivant.

Cocher l’option d’Installation basée sur un rôle ou une fonctionnalité. Puis cliquer sur Suivant.

Sélectionner un serveur du pool de serveurs. Puis cliquer Suivant.

Cocher la case Services Bureau à Distance. Cliquer sur Suivant.

Cliquer Suivant.

Cliquer encore sur Suivant.

Cocher le service Gestionnaire de licences des services Bureau à distance puis cliquer sur Suivant.

Cocher la case Redémarrer automatiquement le serveur de destination si nécessaire. Cliquer sur Installer.

NOTE : Aucun redémarrage n’est requis pour cette opération.

Patienter durant le processus d’installation.

Une fois l’install terminée, cliquer sur Fermer.
