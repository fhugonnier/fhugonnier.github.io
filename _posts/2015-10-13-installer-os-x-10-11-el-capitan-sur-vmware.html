Introduction

 

 

Dans ce tutoriel nous allons voir comment installer le dernier OS d’Apple (OSX El Capitan ) dans une machine virtuel grâce à VMware.

Quelles sont les avantages d’installer El Capitan sur VMware:

  1. Tester l’OS depuis une machine virtualisé avant de faire la mise à jour si vous disposez d’un Mac.
  2. Utiliser un programme qui ne fonctionne que sur OSX.
  3. Utiliser l’os d’Apple si vous n’avez pas les moyens ou l’envie de vous offrir un Mac.

Pour ce tutoriel nous l’avons fait sous forme de vidéo.

 

Prérequis

 

    • Disposez de VMware workstation . La version 12 est recommandé.
    • Téléchargez le pack contenant OSX El Capitan et les fichiers nécessaires à la mise en place : Téléchargez ICI
    • Unlocker pour VMware workstation 12 : Unlocker
    • Disposer d’un processeur 64 bits.
    • Activer la fonction de virtualisation dans le Bios

 

Installation

 

Avant de commencer décompressez le fichier téléchargé grâce à 7Zip.
