Dans cet article nous allons passer à la configuration de domoticz ,ouvrez le navigateur et tapez http://votre.ip.raspberry:8080.

Vous allez arriver sur cette page, pour l’instant aucun module n’est associé.

Pour commencer, aller dans Setup et Application Settings, réglez la langue sur français et indiquer votre localisation grâce au lien.

On descend un peu plus bas, on définit son identifiant et mot de passe de connexion, on peut autoriser des ip du réseau local à se connecter sans authentification c’est assez pratique, on active les onglets que l’on veut.

On configure son compte Email

Exemple pour gmail:

  • serveur: smtp.gmail.com.
  • port: 587.
  • identifiant: ton.adresse.mail@gmail.com.
  • mot de passe: MDP du compte Email.

Dans notification on peut donner la clef API d’une application IOS/Android pour pouvoir gérer Domoticz depuis un Smartphone.

c’est tout pour la configuration.

Installation du rfx433 sur Domoticz

Notre rfxcom est branché sur l’usb du pi ,on va dans configuration, materiel.

On cherche le rfxcom dans la liste déroulante, on laisse le délai d’attente désactiver, on clique sur ajouter et notre box domotique est prête à communiquer en RF433 avec tous nos modules.

Association des modules.

Pour associer un module RF433 à domoticz, il faut aller dans configuration, dispositifs, pour les modules de la gamme chacon de ce type ou celui la avec un bouton d’apprentissage, il faut aller dans interrupteur/ajout manuelle sélectionner votre rfx433, nommez votre module, choisissez switch type: on/off type: arc et choisissez les 2 derniers au hasard mais chaque code corresponde à un module ou un groupe,  appuyiez sur le bouton d’apprentissage du module et dans les 5 secondes Cliquer sur test le module va faire un bruit de claquement pour dire qu’il est associé.

Module variateur.

Pour ajouter un module variateur de type  Chacon 54514, il faut choisir Switcth Type: dimmer Type: AC, Unit Code: 1, même procédure pour l’apprentissage.

  Contacteur de porte sans-fil.

Pour associer un module de type Contacteur CH54781, on va dansConfiguration/dispositif, on ouvre notre capteur, il devrait apparaître dans la liste Cliquez sur la flèche verte pour ajouter un dispositif, nommez le et le tour est joué.

Sonde de température.

Je dispose de 2 sondes de température type Oregon, elle relève que la température mais d’autre modèle celle-ci relève la température et l’humidité.

On reste dans configuration/Dispositif, cette fois-ci on cherche notre sonde de température.

On Clique sur la flèche verte, nomme notre sonde même opération pour la seconde ainsi que tous nos autres modules.

Résultat :

Pour finir, on se retrouve avec notre jolie petite interface avec nos Interrupteurs et nos sondes de Température.

Cliquez sur les logs pour voir l’historique des modules, et température.

Scénarios

Pour créer une scène, on va aller dans Scénario/Crée un scénario. On va créer un scénario maison entière, on ajoute toutes les devises qu’on veut grouper et éteindre ou allumer d’un clique.

On crée un scénario pour associer la télécommande, on le nomme comme on veut.Exemple: cuisine pour associer la lumière de ma cuisine à un bouton de la télécommande.

On Clique sur Composer un code, on appuie sur le bouton de la télécommande, on choisit sa devise et on recommence pour toutes les touches de la télécommande.

On définit des favoris dans les modules pour les réunir sur l’accueil afin d’avoir une vue d’ensemble.
